A mobile-style pattern lock built with Python and Tkinter.
Features a neon-themed grid, dot animations, gradient lines, enrollment, unlocking, limited attempts, and reset functionality.

Features

Enroll a new pattern (first-time setup).

Unlock using the enrolled pattern.

Limit on incorrect attempts (default 5).

Reset pattern option to start over.

Dot pop animation when touched.

Neon gradient lines connecting dots while drawing.

Prevent unlocking before enrollment.

Works on any standard Python environment with Tkinte
